# Setup
1. Create a new python virtual environment:
`python3 -m venv .`

2. Activate newly created python virtual environemnt:
`source bin/activate`

3. Install required packages:
`pip install -r requirements.txt`

4. To open jupyter notebook from the terminal:
`jupyter notebook`
